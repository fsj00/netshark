## 项目概况

NetShark 是一个基于 Web 的 PCAP 文件在线分析工具，使用 sharkd 提供数据包解析功能。

- **后端**: Go + Gin 框架
- **前端**: 原生 HTML/JS/CSS
- **解析引擎**: sharkd (Wireshark JSON-RPC)
- **代码行数**: 4,457 行
- **文件数量**: 33 个
